# Valanga

A Ruby client for [REFLEC BEAT](http://p.eagate.573.jp/game/reflec/groovin/p/index.html)

## Usage

## Dependency

* [PhantomJS](http://phantomjs.org/)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'valanga'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install valanga
